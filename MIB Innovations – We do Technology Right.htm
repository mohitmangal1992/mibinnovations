MIB Innovations

Analytics:
  • IBM Cloud Pak for Data (includes Big Data)
  • Apache Spark based custom solutions
  • Complex Reports (Crystal/Cognos)

Use cases where we can help:

  • You have a lot of structured data in various applications and a lot of archived data. It needs to be analyzed and joined together for analytics. This can be done using IBM Cloud Pak for Data. It is an IBM solution that has the various options in the data layer, processing layer and the visualization layer. Any combinations of these can be used to build a solution.
  • Along with the structured data, there is also a lot of un-structured and semi-structured data that is also required for analytics. We can extract the data from text documents, pdf files, emails and other data stores and make it ready for performing analytics.
  • You have a business process that is still not automated or requires a lot of manual repeated work. Using a Robotic Process Automation and OCR technology we can automate the process. This means the information can be read from pdf documents and entered into Enterprise applications.
  • You have a good application but reporting needs enhancement. We can build a complete reporting solution.

Video:
  • Host videos using IBM streaming platform
  • Facial Recognition using IBM Intelligent Video Analytics platform
  • Provide Close Captioning in various languages
  • Make custom videos using Adobe Premier Pro, Photoshop etc.

Use cases where we can help

  • Your HR department has on-boarding videos. Marketing department has product videos. Your Operations department has learning videos for various processes. These can be hosted and streamed on the IBM streaming platform on the cloud even if you have thousands of employees at a location.
  • You want to perform facial recognition in videos by searching a particular person in a video. This can be done using IBM Intelligent Video Analytics platform.
  • You want to provide closed captioning for your videos in multiple languages. This is done using Watson Translation that automatically converts speech to English text. You can proof read and update the text and Watson learns the changes. This text can then be translated to other languages. Proof read and update the translated text. Watson will learn the changes.

Infrastructure:
  • Install and maintain Linux/Windows servers and VMs/Containers
  • Install, maintain and tune Oracle/SQL Server/MySql/MongoDB
  • Install and maintain applications using Dockers/Kubernetes

You want to run your applications on a high performance infrastructure without the huge upfront costs. Your applications should be accessible to your mobile employees.

Use cases where we can help

  • Migration of your applications from local infrastructure to a Virtual Public cloud
  • Your data is very confidential and you do not feel very secure on the Public cloud. We can build you a Private cloud and move your applications to the Private cloud.
  • You want to have a mix of Public and Private cloud for your applications. We can do the necessary security configuration.
  • You want to have a mix of Local, Public and Private cloud for your applications. We can do the necessary security configuration.
  • You have a large Oracle/SQL Server/MySQL/MongoDB databases that needs continuous DBA support or tuning or migration
